## The Lár Project

**[GitHub Repository](https://github.com/snath-ai/lar) | [Documentation](https://docs.snath.ai)**

**Lár** (Irish for "core") is the open-source flight recorder for AI agents. It provides a deterministic, auditable execution engine for building complex workflows without the "black box" magic.

*   **Deterministic**: Reproducible execution paths.
*   **Auditable**: Every step is logged to an immutable JSON flight plan.
*   **Air-Gap Capable**: Runs completely offline.
